# /brainstorm

Read `.agent/skills/brainstorming/SKILL.md` and follow it exactly.

The brainstorming skill is a checklist — start at step 1 (explore project context), proceed one step at a time, and get user approval before transitioning to implementation.

If the user has not yet stated what they want to brainstorm, ask: "What idea or feature do you want to explore?" — then begin the skill checklist.
